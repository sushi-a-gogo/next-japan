👋 Welcome to "Next Japan"

A full-stack demo app which exists in an imaginary world inspired by Japanese Animation, allowing imaginary users to register for special events hosted at various locales in Japan.

This is a self-built portfolio project by me, showcasing modern full-stack engineering with:

  • 🧩 Angular v20 frontend
  • 🚀 Node.js + Express backend
  • 🗃️ MongoDB for persistent event storage and dynamic content management
  • ☁️ Live deployment on Render handling
  • 🤖 OpenAI+Grok API integration for smart content features
  • ⚙️ Full environment config, CORS handling
  • 🌐 Cloudflare Images integration for optimized, globally cached image delivery
  • 🖥️📱 Responsive design supporting both desktop and mobile
  • 🚀 SSR for optimized performance and SEO
