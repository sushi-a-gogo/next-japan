About this App
👋 Welcome to "Next Japan"

A full-stack demo app which exists in an imaginary world inspired by Japanese Anime, allowing imaginary users to register for special events hosted at various locales in Japan. This is a self-built portfolio project by Joe Svatek, showcasing modern full-stack engineering with:

  • 🧩 Angular v20 frontend
  • 🚀 Node.js + Express backend
  • ☁️ Live deployment on Render (free tier) with cold start handling
  • 🤖 OpenAI API integration for smart content features
  • ⚙️ Full environment config, CORS handling, and simulated data APIs

⏳ Note: Initial load may take up to a minute due to free-tier backend startup. A loading screen will appear during this time.
