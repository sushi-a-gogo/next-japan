arrow_back_ios_new About this Project
Next Japan is a full-stack web application showcasing signal-driven Angular v20 UI, MongoDB APIs, and responsive design. It simulates an imaginary world inspired by Japanese Animation, allowing imaginary users to register for special events hosted at various locales in Japan. The project was designed, architected, coded, and deployed entirely from scratch by me.
Key Skills
  • 🧩 Angular v20 with standalone components and signal-based state management.
  • 🔮 Zoneless architecture: Removed Zone.js for optimal performance.
  • 👾 Node.js + Express backend
  • 🗃️ MongoDB for persistent event storage and dynamic content management.
  • 📡 RESTful API with rate limiting, CORS handling
  • 🤖 Integration with the OpenAI and Grok APIs for enhanced features.
  • 🌐 Cloudflare Images integration for optimized, globally cached image delivery.
  • ☁️ Cloud deployment hosted on Render.
  • 📱 Responsive design with selectable light and dark modes.
  • 🚀 SSR for optimized performance and SEO.
  • 🔒 JWT-based authentication with auto-refresh.
