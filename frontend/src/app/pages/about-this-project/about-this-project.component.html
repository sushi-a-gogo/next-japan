👨‍💻 About This Project
Next Japan is a full-stack portfolio project I built to showcase real-world engineering skills, explore new technologies, and stay engaged while on the job hunt.

It simulates an imaginary world inspired by Japanese Anime, allowing imaginary users to register for special events hosted at various locales in Japan.

Designed, architected, coded, and deployed entirely from scratch using:
  • ⚙️ Angular v20 with modular routing, components, and services
  • 🌐 Node.js + Express backend hosted on Render
  • 📡 RESTful API with rate limiting, CORS handling, and mocked JSON data
  • 🌐 Cloudflare Images integration for optimized, globally cached image delivery
  • 🤖 Integration with the OpenAI and Grok APIs for enhanced features
  • ☁️ Cloud deployment with cold start loading overlay
  • 🧪 Environment-specific configs, error handling, and responsive UI
